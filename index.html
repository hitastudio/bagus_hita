Tentang Saya

 Halo, saya Bagus Hita Bayu Dawara, mahasiswa Program Studi Teknik Rekayasa Perangkat Lunak (TRPL) di Politeknik Negeri Lampung (Polinela).

 Saya memiliki ketertarikan yang kuat pada game development sejak masa SMP, dan sejak itu terus mengembangkan kemampuan saya di bidang ini secara mandiri.

 Meskipun di bangku kuliah saya belajar pengembangan perangkat lunak secara umum, saya tetap fokus mengeksplorasi pembuatan game sebagai passion utama. Saya aktif mencoba berbagai proyek, mulai dari membuat prototipe game, mempelajari mekanik gameplay, hingga menciptakan pengalaman bermain yang menyenangkan.

 Selain itu, saya sering mengikuti lomba-lomba game, baik secara individu maupun tim, untuk mengasah kemampuan teknis dan kreativitas. Salah satu pengalaman berharga saya adalah bekerja bersama Gemilang Studio, di mana saya belajar banyak tentang proses kolaborasi tim, mulai dari perencanaan hingga implementasi game.

 Saya percaya bahwa kombinasi minat sejak dini, kemampuan belajar mandiri, dan pengalaman proyek nyata membuat saya terus berkembang sebagai seorang pengembang game. Saya selalu bersemangat menghadapi tantangan baru, berinovasi, dan menciptakan game yang menarik, seru, dan berkesan bagi pemain.
