Tentang Saya

 Halo, saya Bagus Hita Bayu Dawara, mahasiswa Program Studi Teknik Rekayasa Perangkat Lunak (TRPL) di Politeknik Negeri Lampung (Polinela).

 Saya memiliki ketertarikan yang kuat pada game development sejak masa SMP, dan sejak itu terus mengembangkan kemampuan saya di bidang ini secara mandiri.

 Meskipun di bangku kuliah saya belajar pengembangan perangkat lunak secara umum, saya tetap fokus mengeksplorasi pembuatan game sebagai passion utama.

 Selain itu, saya juga aktif mengikuti berbagai lomba game, baik secara individu maupun tim, untuk mengasah kemampuan teknis sekaligus kreativitas. Saya juga senang mendokumentasikan proses pengembangan game yang saya kerjakan dan membagikannya dalam bentuk konten, sehingga sekaligus berperan sebagai content creator yang fokus pada perjalanan dan progres pembuatan game.
